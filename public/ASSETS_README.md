# Forever Bali Villas — Asset Placement Guide

Drop your images and videos into the folders below.
Recommended formats: WebP or JPG for images, MP4 (H.264) for videos.

## NAMING TIPS
- Use lowercase letters and hyphens only (no spaces, no underscores)
- Recommended image max-width: hero=1920px, cards=800px, gallery=1200px
- Recommended image formats: WebP (best) or JPG
- Videos: MP4 H.264, max 10MB for web performance
